Привет, это команда Consta!

v1

Consta font replacer — плагин для автоматизированной замены шрифтов в макетах Figma. Пока работает только для пары Segoe UI -> Inter. Замена производится постранично (pages), а не во всем файле сразу.

Во время работы плагина лучше не трогать макеты и просто подождать
